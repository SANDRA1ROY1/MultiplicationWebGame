Rules

1.

press start to initiate the game

2.

player will be displayed with an answer which is a result of equation with one of the enemy

3.

you should press Up or Down arrow keys to reach the same row of the corresponding enemy

4.

press space key to defeat the enemy with your answer

5.

if you defeat the enemy with the correct answer, you would be up by a level

6.

if you try to defeate with a wrong answer, a new set of enemies are generated and your level remains the same

7.

if the timer expires or any one of the enemy reaches right side, player loses the game

8.

if player successfully defeats all enemies within the time limit, the player reaches level 5 and wins the game
